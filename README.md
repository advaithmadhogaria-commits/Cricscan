# 🏏 CricScan – AI Cricket Scorer

A fully-featured PWA cricket scoring app with AI camera detection.

## 🚀 Deploy to Vercel in 3 Steps

### Step 1 – Create a GitHub account (if you don't have one)
Go to https://github.com and sign up for free.

### Step 2 – Upload this project to GitHub
1. Go to https://github.com/new
2. Name the repo: `cricscan`
3. Click **Create repository**
4. Click **uploading an existing file**
5. Drag and drop ALL the files from this zip into the browser
6. Click **Commit changes**

### Step 3 – Deploy on Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click **Add New Project**
3. Select your `cricscan` repo
4. Vercel auto-detects Vite — just click **Deploy**
5. In ~1 minute you get a link like: `https://cricscan.vercel.app`
6. **Share that link with anyone!** Works on all phones, no login needed.

## ✨ Features
- Manual scoring: 0,1,2,3,4,6 + Wide, No Ball, Bye, Leg Bye
- AI Camera detection — draw zones on pitch, app auto-scores
- Full scorecard with batting & bowling stats
- Wicket flow with dismissal types
- Strike rotation & bowler selection
- Over-by-over ball tracker
- Fall of wickets
- PWA — users can "Add to Home Screen" for native app feel
- Works on ALL phones (Android & iPhone)

## 🛠 Local Development
```bash
npm install
npm run dev
```
